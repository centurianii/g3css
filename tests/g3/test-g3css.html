Evaluator: Test g3.css classes

[edit]

We prepared the following files:

  1. test-g3css.html the basic evaluator's GUI enhanced with the batch tests
  2. test-g3css-stub1.html a test stub or, a framed page that contains linked, imported and embedded style sheets and linked and embedded script commands
  3. test-g3css-stub-link1.css the first linked css file in the head of test-g3css-stub1.html
  4. test-g3css-stub-link2.css an imported style sheet in test-g3css-stub-link1.css
  5. test-g3css-stub1.js the first linked javascript file in the head of test-g3css-stub1.html*
  6. test-g3css-stub2.js the second linked javascript file in the body of test-g3css-stub1.html*

*It was needed to conduct some tests on the clone frame methods of the g3.evaluator object which is now at version 0.1.3.
